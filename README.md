# CHIVE

## Prerequisites

This project requires Node.js (which includes npm).

### Install Node.js

Choose one option:

1. Official installer (recommended for most people)
	- Go to https://nodejs.org/
	- Download the LTS version
	- Run the installer and keep the default options

2. Windows package manager (if you use it)
	- `winget install OpenJS.NodeJS.LTS`

After installing, open a new terminal and verify:

- `node -v`
- `npm -v`

If both commands print versions, you are ready.

## Development

1. Run `npm install`
2. Run `npm run dev`
3. Open the local Vite URL shown in the terminal
